# Opacity

Read about how you can use **Opacity** to make your content appear or disappear.

| Class | Properties      |
| ----- | --------------- |
| o-10  | `opacity: 0.1;` |
| o-20  | `opacity: 0.2;` |
| o-30  | `opacity: 0.3;` |
| o-40  | `opacity: 0.4;` |
| o-50  | `opacity: 0.5;` |
| o-60  | `opacity: 0.6;` |
| o-70  | `opacity: 0.7;` |
| o-80  | `opacity: 0.8;` |
| o-90  | `opacity: 0.9;` |
| o-100 | `opacity: 1;`   |
