## Installation

To install our cli you can use the following commands

<code-group>
<code-block title="YARN">
```bash
yarn global add hypestyle-cli@latest
```
</code-block>

<code-block title="NPM" active>
```bash
npm install -g hypestyle-cli@latest
```
</code-block>
</code-group>

or

```bash
npx hypestyle-cli@latest
```
