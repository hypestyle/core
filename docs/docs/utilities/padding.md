# Padding

In hypestyle css we have a set of utilities to add padding to an element.

### Padding

| Class | Properties          |
| :---- | :------------------ |
| `p-0` | _padding: 0;_       |
| `p-1` | _padding: 0.75rem;_ |
| `p-2` | _padding: 1.5rem;_  |
| `p-3` | _padding: 3rem;_    |
| `p-4` | _padding: 4.5rem;_  |
| `p-5` | _padding: 6rem;_    |
